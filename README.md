## PETUNJUK INSTALLASI

```bash

gitclone https://github.com/FarizAditya/BVCScripsi.git

cd BVCScripsi

composer install

ubah file nama fileenv jadi .env

```
## INSTALASI APLIKASI

```bash

php artisan key:generate

php artisan migrate --seed

```
